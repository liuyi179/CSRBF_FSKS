# CSRBF_FSKS
This project contains MATLAB code for solving the three-dimensional three-temperature radiation diffusion equation using the compactly supported radial basis function (CSRBF) method.
The project contains the following files: the main program file CSRBFs_FSKS.m; 
the utility function files DifferenceMatrix.m, DistanceMatrix.m, and DistanceMatrixCSRBF1.m; 
the project documentation README.md;
the cylindrical domain node data files yuanzhu_27.mat, yuanzhu_125.mat, yuanzhu_729.mat, and yuanzhu_4913.mat.
